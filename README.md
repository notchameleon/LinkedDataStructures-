# Linked Data Structures Portfolio

A comprehensive collection of linked data structure implementations in Java.

## 📁 Project Structure

### Linked List
- **LinkedList.java** - Basic linked list implementation
- **LinkedUnorderedList.java** - Unordered list using links
- **ArrayOrderedList.java** - Ordered list (array-based, for comparison)
- **Node.java** - Node class for linked structures

### Linked Stack  
- **LinkedStack.java** - Stack implementation using linked nodes
- **StackADT.java** - Stack interface
- **Lab5App.java** - Stack application/demo
- **Card.java** & **Node.java** - Supporting classes

### Linked Queue
- **LinkedQueue.java** - Queue implementation using linked nodes
- **QueueADT.java** - Queue interface
- **Lab6App.java** - Queue application/demo
- **Product.java** & **Node.java** - Supporting classes

## 🚀 How to Use
1. Clone repository: \git clone https://github.com/notchameleon/linked-data-structures.git\
2. Open in Eclipse/IntelliJ
3. Run test applications in each package

## 📊 Implementations Overview
| Structure | File | Description |
|-----------|------|-------------|
| Linked List | LinkedList.java | Basic singly linked list |
| Linked Stack | LinkedStack.java | LIFO structure with nodes |
| Linked Queue | LinkedQueue.java | FIFO structure with nodes |

## 🎯 Learning Objectives
- Understand node-based data structures
- Implement common operations (add, remove, search)
- Compare linked vs array implementations
